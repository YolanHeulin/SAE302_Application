# SAE302 - Application Android

## Description
Application Android développée dans le cadre de la SAE302.
Elle permet la planification d'interventions technicien.

## Objectifs
- Comprendre la gestion opérationnelle d’un planning
- Organiser les tâches par priorité
- Concevoir l’écran d’une intervention : date, site, actions, statut

## Technologies
- Android Studio
- Java

## Installation
1. Cloner le dépôt
2. Ouvrir le projet avec Android Studio
3. Lancer l’application sur un émulateur ou un téléphone
